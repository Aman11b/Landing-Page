# The Green Landing Page

This is a simple landing page project created as part of [The Odin Project](https://www.theodinproject.com/) curriculum. It uses HTML and CSS (Flexbox) to build a clean, responsive landing page layout.

## Features
- Header section with navigation links styled using Flexbox
- Responsive layout
- Minimal and clean design

## Technologies Used
- HTML
- CSS (Flexbox)
